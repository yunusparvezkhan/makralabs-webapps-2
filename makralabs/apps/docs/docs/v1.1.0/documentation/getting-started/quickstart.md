# Quickstart

Launch the docs experience with a hierarchy that scales from onboarding to reference material.

## Overview

This guide introduces the versioned docs structure, tab-level navigation, and nested URLs used across the site.

## What you will do

- Open the documentation tab from the top navigation.
- Follow the left sidebar through the `getting-started` chapter.
- Use full document URLs when sharing links with teammates.

## Why this structure works

Nested paths make the information architecture obvious at a glance and keep related pages grouped together.
