# Advanced Scraping Guide

Capture feature launches in a dedicated section so product updates are easy to browse and easy to archive later.

## Overview

This page is a good home for release-stage guidance, migration notes, and examples that are still evolving.

## Suggested sections

- What changed and who should care.
- New API behavior or configuration knobs.
- Rollout notes and backward-compatibility details.
