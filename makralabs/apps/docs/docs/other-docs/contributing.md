# Contributing

Makra Docs are designed to be maintained as Markdown.

## How to propose a change

1. Edit or add a Markdown file under `apps/docs/docs`
2. Update `apps/docs/docs-config.yaml` to wire the page into navigation
3. Open a PR with a short description of the change

## Style guidelines

- Start each page with a single `#` title
- Keep examples runnable
- Prefer short sections with clear headings
